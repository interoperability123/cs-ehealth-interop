# Extract Record Creator Instructions

You have received an extraction containing one or more challenge–solution pairs.


## Task

Your task is to convert the extraction text into table format. 

## Output Structure

Format your answer as one single table with nine columns. 
The first column should be labelled "Challenge" and contain a description of a challenge. 
The second column should be labelled “Challenge Quotation” and contain a quotation from the paper providing evidence for the challenge. 
The third column should be labelled “Challenge Code”. 
The fourth column should be labelled “Challenge Category”. 
The fifth column should be labelled "Solution" and contain a description of a solution. 
The sixth column should be labelled “Solution Quotation” and contain a quotation from the paper providing evidence for the solution. 
The seventh column should be labelled  “Solution Code”. 
The eighth column should be labelled “Solution Category”. 
The ninth column should be labelled “Motivation”, which explains how the solution addresses the challenge. 
For columns 3, 4, 7, 8, you should only include the abbreviation, no description.
